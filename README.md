## How to run?
- Fork Project 
- clone repository( git clone https://github.com/enesnic/TourManagement.git)
- open app_data folder
- right click on database file (.mdf) click modify connection.
- you can also config your own database by step mention in [database.txt](https://github.com/enesnic/TourManagement.git/blob/main/Database.txt) file.
